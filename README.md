# 🤖 AI Agent using LangChain

## 📌 Overview
This is an AI-powered Agent built using LangChain and OpenRouter LLMs.  
It can answer questions using Wikipedia, Google Search (SerpAPI), and LLM reasoning.

---

## ⚙️ Features
- 🧠 AI-powered responses (GPT via OpenRouter)
- 🌐 Wikipedia search
- 🔎 Google search (SerpAPI)
- 🤖 Automatic tool selection
- 💬 Interactive chatbot (CLI)

---

## 🏗️ Tech Stack
Python, LangChain, OpenRouter API, SerpAPI, Wikipedia API

---

```bash
git clone https://github.com/nikitabiradar24/AI-Agent.git
